# AtCoder

・AtCoder 記録用

## online-judge-tools

https://github.com/online-judge-tools/oj

### Example

1. Login

```
$ oj login -u {user_id} -p {password} "https://atcoder.jp/"
```

2. Download

```
$ oj download http://agc001.contest.atcoder.jp/tasks/agc001_a
```

3. Test

```
$ oj t -c "python3 main.py"
```

4. Submit

```
$ oj s main.py
```
